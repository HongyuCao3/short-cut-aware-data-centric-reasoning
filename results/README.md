# results/ — Experiment Output Tables

This directory contains plain-text result tables generated by `src/visualize.py` after running `run_all.py`.

## Files

| File | Content |
|------|---------|
| `table1.txt` | Main results — all 13 methods across 3 synthetic datasets |
| `table2.txt` | Ablation — SART component contributions (reweighting vs. gradient surgery) |
| `table3.txt` | Real-world results — GSM8K and MATH datasets |
| `table4.txt` | Per-dataset breakdown with gradient alignment scores |

## How to Reproduce

```bash
# Server scale, all datasets
EXPERIMENT_SCALE=server DATASET_TYPE=all python3 run_all.py
```

Tables are written automatically to this directory when the run completes.

## Key Findings

**Main results (averaged across 3 synthetic datasets, server scale):**

| Method | Accuracy | Robustness |
|--------|----------|------------|
| Standard FT | 59.6% | 1.9% |
| Group DRO | **78.2%** | **48.2%** |
| **SART (Ours)** | 75.8% | 39.9% |

**Ablation (SART components):**

| Configuration | Accuracy | Robustness | Grad. Align. |
|---------------|----------|------------|--------------|
| Standard FT | 59.6% | 1.9% | −0.07 |
| Reweighting Only | 64.7% | 13.1% | — |
| Gradient Surgery Only | 81.0% | 51.3% | — |
| **Full Method** | 75.8% | 39.9% | **+0.10** |

Gradient Surgery is the primary mechanism; the full method achieves the best gradient alignment.
